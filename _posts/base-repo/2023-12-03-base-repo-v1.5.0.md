Today, we like to announce the new release v1.5.0 of base-repo. This release mainly contains the upgrade to Spring Boot 3, but also fixes some minor issues. 
If you face any new issues, please tell us via the [Issue Tracker](https://github.com/kit-data-manager/base-repo/issues).

##  New Features
* Upgrade to Spring Boot 3
* Java 17 is now mandatory
* Indexing support now requires Elastic 8+

## Bugfixes
* Correctly ignore user-provided primary keys (https://github.com/kit-data-manager/base-repo/pull/131)
* Fixed issues while providing escaped slashes in request URL
